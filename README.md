# WASSII Billiard Clock

ビリヤード対戦用の **スコアボード兼タイムクロック** をブラウザだけで使えるようにしたシンプルな Web アプリです。  
スマホ・タブレット・PC のブラウザで動作し、インストール不要でそのまま利用できます。

## 主な機能

- 2人対戦用のチェスクロック風タイマー
- プレイヤーネームの編集（`PLAYER 1` / `PLAYER 2` をタップして変更）
- スコア管理（＋／−ボタン）
- 持ち時間（分）の変更機能（1〜120分）
- 残り時間が 0 になったあと、30秒のショットクロック（秒読み）
- 一時停止／再開ボタン
- 誤タップ防止の「リセット確認ダイアログ」
- 画面サイズに応じた自動レイアウト（スマホ縦・横／タブレット／PC対応）

※現時点の公開版は **ビープ・音声なし** のシンプル版です（今後アップデート予定）。

## 使い方（ユーザー向け）

1. ブラウザでこのアプリの URL（GitHub Pages など）を開きます。
2. 必要に応じて、プレイヤーネームをタップして名前を変更します。
3. 「持ち時間（分）」の `- / +` ボタンで、1人あたりの持ち時間を設定します。  
   設定を反映するには「リセット」ボタンで初期化してください。
4. 先行プレイヤーは、自分側の大きなパネル（時計エリア）をタップしてゲームを開始します。
5. ターンが終わったら、自分側のパネルをもう一度タップすると、相手プレイヤーの時間が動き出します。
6. 残り時間が 0 になると、そのプレイヤーは 30 秒のショットクロック（秒読み）に入ります。
7. トイレなどで一時的に止めたい場合は「一時停止」ボタンを押します。再開するときはもう一度押します。
8. 試合を最初からやり直したい場合は「リセット」ボタンを押します。  
   リセット時には確認ダイアログが表示されます。

## スマホでアプリのように使う（ホーム画面に追加）

### iPhone（Safari）

1. Safari でアプリの URL を開く  
2. 画面下部の共有アイコンをタップ  
3. 「ホーム画面に追加」をタップ  
4. 名前を確認して「追加」

### Android（Chrome）

1. Chrome でアプリの URL を開く  
2. 右上のメニュー（三点）をタップ  
3. 「ホーム画面に追加」または「インストール」をタップ  
4. 名前を確認して追加

## 開発者向け（ローカルでの動作確認）

このプロジェクトは、単一の `index.html` で完結する静的な Web アプリです。

### 必要なもの

- 任意の Web ブラウザ（Chrome / Edge / Firefox / Safari など）

### ローカルで開く

1. このリポジトリをクローンまたは ZIP ダウンロードします。
2. `index.html` をダブルクリックしてブラウザで開きます。

特別なビルドやサーバー構築は不要です。

---

## GitHub Pages での公開メモ（オーナー向け）

1. GitHub のリポジトリで `Settings` → `Pages` を開きます。
2. 「Build and deployment」の `Source` を  
   `Deploy from a branch` に設定し、
   `Branch` は `main`（または `master`）を選択して保存します。
3. 数分待つと、  
   `https://ユーザー名.github.io/リポジトリ名/`  
   のような URL で公開されます。

この URL を店内 POP や QR コード、SNS のプロフィールなどに使うことで、誰でもブラウザからアクセスできます。
